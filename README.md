# 🚀 MERN Blog Application

A full-stack blog platform featuring role-based access, high-end animations, and seamless MongoDB Atlas integration.

## 📁 Project Structure

- **[backend/](./backend)**: Node.js/Express server with MongoDB Atlas integration.
- **[frontend/](./frontend)**: React/Vite application with Framer Motion and Tailwind CSS.

## 🌟 Key Features

- **Authentication**: Secure JWT-based login and registration with role differentiation (User, Author, Admin).
- **Dashboard**: Interactive dashboards for viewing and managing articles.
- **Article Management**: Full CRUD operations for Authors, including soft-delete and restore functionality.
- **Modern UI**: Premium design with fluid page transitions and stagger animations.
- **Production Ready**: Configured for deployment on Render with centralized API management.

## 🛠️ Quick Start

### 1. Backend Setup
```bash
cd backend
npm install
# Add your .env file
npm start
```

### 2. Frontend Setup
```bash
cd frontend
npm install
# Add your .env file
npm run dev
```
